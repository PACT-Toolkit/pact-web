# Canonical Spec Examples

Load this when starting a new spec and you want to model it on an exemplary existing one. Pick the closest match, read it end-to-end, then adapt.

| Spec | What to learn from it |
| ---- | --------------------- |
| `src/app/efaktura/test/efaktura.spec.ts` | **Gold standard** — multi-view a11y, tabs, side sheets, search. |
| `src/app/direct_debit/test/direct_debit.spec.ts` | Complex multi-flow spec with several distinct user journeys. |
| `src/app/transaction_search/test/transaction_search.spec.ts` | Filters, date pickers, nested `describe` blocks. |
| `src/app/file_request/test/file_request.spec.ts` | File uploads and toast assertions. |
| `src/app/international_transfer/test/international_transfer.spec.ts` | Multi-step wizard, challenge / OOB flows. |
