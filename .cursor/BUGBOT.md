# BugBot Custom Instructions

**Only apply these checks if the PR includes changes to `app/locales/`.**
If no locale files are modified, skip this entire review.

---

## Translations

When a PR touches files under `app/locales/{da,en,no,sv}/*.json`, apply the rules in `.agents/skills/translate/SKILL.md`.

## User-facing copy (English)

When a PR adds or modifies values in `app/locales/en/*.json`, apply the rules in `.agents/skills/writing-for-smb/SKILL.md`.
